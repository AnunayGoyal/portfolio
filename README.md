## âš™ï¸ Setup & Customization

If you want to clone this for your own use:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/AnunayGoyal/portfolio.git
    cd portfolio
    ```

2.  **Update the Contact Form**
    Open `contact.html` and replace the Formspree URL with your own:
    ```html

    ```
